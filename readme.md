# git-repo

**Work in progress**

## Installation

**Homebrew (macOS)**

```
brew tap josa42/homebrew-git-tools
brew install git-repo
```

**Other**

```
go get github.com/josa42/git-repo
```

## Usage

```
Usage:
  git-repo home
  git-repo commits
  git-repo issues
  git-repo prs
  git-repo pr
  git-repo compare <older-revision> [<newer-revision>]

Options:
  -h --help          Show this screen.
  --version          Show version.
```
